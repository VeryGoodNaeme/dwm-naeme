# dwm-naeme
Installation
```bash
git clone https://github.com/verygoodnaeme/dwm-naeme
cd dwm-naeme
sudo make install
```
If you use a display manager add the following to /usr/share/xsessions/dwm.desktop
```bash
[Desktop Entry]
Encoding=UTF-8
Name=Dwm
Comment=Dynamic window manager
Exec=dwm
Icon=dwm
Type=XSession
```
